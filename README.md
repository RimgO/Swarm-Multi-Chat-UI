# Swarm Multi Chat UI
Streamlitを使用したマルチエージェントチャットアプリケーション。複数のAIエージェントと同時に会話できます。

## 機能

- 複数のAIエージェントとの同時チャット
- ユーザー認証システム  
- エージェント間でのコンテキスト共有
- ユーザー間でのコンテキスト共有
- レスポンシブなUI

## 利用可能なエージェント

- 聴き上手なカウンセラー: 共感的な聴き手として対話
- 的確なアドバイザー: 問題解決のための具体的なアドバイスを提供  
- 心理分析の専門家: 心理学的な視点から分析

## セットアップ

1. 必要なパッケージのインストール:
```bash
pip install -r requirements.txt

## 使用方法

1. サイドバーからユーザー名とパスワードでログイン
2. 利用したいAIエージェントを選択
3. チャットウィンドウでエージェントと対話
4. 必要に応じて他のユーザーとコンテキストを共有

## 注意事項

- エージェントのレスポンスは100文字以内に制限されています
- コンテキスト共有は最新の3つのメッセージに限定されます
